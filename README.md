## 中国矿业大学（北京）论文模板

> 最好的模板就是一篇基于模板的完整论文， 其所包含的公式、图片、表格等不再是 demo， 只有把真实场景中的使用形式呈现给使用者看， 才能真的让使用者少淌些坑。
